# Watermark

Minimal static web app for adding text watermarks (with adjustable opacity/angle and quick date insertion) to images and videos, plus black redaction boxes for images before sharing ID scans or other sensitive documents.

All data stays on your machine.

Don't want to trust a random website with personal data? I feel you. Just download the repo and run `python -m http.server 8000` or something similar to run it directly on your machine.
